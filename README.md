# 广域 RDMA 与分布式训练系统优化 (Wide-Area RDMA Research)
## 项目目标
本仓库用于记录关于 **高性能网络（HPN）** 与 **分布式训练框架** 的协同设计研究。
核心目标是针对 **广域网（WAN）** 环境下的高延时、高丢包和带宽抖动特征，寻找 RDMA 通信协议的优化空间及创新点。

做广域RDMA优化 基于此做上层推理or训练的优化

**复现**：把代码跑通之后，了解每一部分是做什么的，为什么要这么做，input是什么，output是什么，然后和论文里头的方法，公式细节对起来
